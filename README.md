# sandlerz/skills

My personal collection of [Agent Skills](https://agentskills.io) for Claude Code and other AI coding agents.

## Install all skills

```bash
npx skills add sandlerz/skills --all -g -a claude-code
```

## Install a specific skill

```bash
npx skills add sandlerz/skills --skill <skill-name> -g -a claude-code
```

## List available skills

```bash
npx skills add sandlerz/skills --list
```

## Skills

| Skill | Description |
|-------|-------------|
| [git-flow](skills/git-flow/SKILL.md) | Git + GitHub workflow: save (add/commit/push), update branch, create PRs, sync |

## Compatible agents

These skills follow the open [Agent Skills specification](https://agentskills.io) and work with Claude Code, Codex, Cursor, and more.
